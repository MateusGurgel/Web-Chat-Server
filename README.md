## Run Locally

Clone the project

```bash
  git clone https://github.com/MateusGurgel/Web-Chat-Client
```

Go to the project directory

```bash
  cd Web-Chat-Client || Web-Chat-Server
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```

Open http://localhost:3000 with your browser to see the result!
